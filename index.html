WELCOME

We provide the basic information and simulation about simplified version of solar system using web technologies. THUS, WebGL and external libraries were used for API(s) that are required to create 3D intensive works. For more information keep scrolling down.

Objectives & Goals


  • To provide the visualization of our solar system as simple as possible.

Requirements

  • WebGL supported browser is required, since the simulation is entirely based on 3D features.
  • latest version of FireFox is recommended
  • Not compitable with display resolution under 1080 x 720
